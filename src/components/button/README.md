# ButtonComponent

## Использование
```html
<button-component text="текст" size="small" color="red" is-active="false">
</button-component>
```

## Атрибуты

- `text`: текст внутри кнопки;

- `size`: размер кнопки.
        Значения:
        - `small`: маленькая кнопка,
        - `medium`: средняя кнопка,
        - `large`: большая кнопка;

- `color`: цвет кнопки.
        Значения:
        - `red`: красная кнопка,
        - `orange`: оранжевая кнопка.

- `is-active`: hover эффект кнопки.
